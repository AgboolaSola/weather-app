# Weather App 🌤️

A responsive weather app that shows real-time weather
and a 5-day forecast for any city in the world.

## Live site

[View live](https://agboolasola.github.io/weather-app/)

## Features

- Search any city for live weather data
- Current temperature, humidity and wind speed
- 5-day forecast using filtered API data

## Built with

- HTML5 and CSS3 (CSS custom properties, Grid, Flexbox)
- Vanilla JavaScript ES6+
- OpenWeatherMap API
- async/await
- Array methods: map, filter, join

## JS concepts practised

- async/await and try/catch
- Array destructuring on API response data
- Template literals for dynamic HTML and URLs
- Optional chaining on nested API objects
- map() to render forecast cards
- filter() to extract one reading per day
